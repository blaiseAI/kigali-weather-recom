KIGALI WEATHER TRACKER
======================

Quick Start Guide
----------------
1. Install dependencies: npm install
2. Add OpenWeather API key to .env.local
3. Run development server: npm run dev
4. Open http://localhost:3000

Features
--------
* Real-time weather updates
* Air quality monitoring
* 5-day forecast
* Smart weather alerts
* Activity recommendations
* Dark mode support
* Mobile responsive

API Key Setup
------------
Create .env.local file with:
OPENWEATHER_API_KEY=your_api_key_here

Tech Stack
----------
- Next.js 14
- TypeScript
- Tailwind CSS
- OpenWeather API
- React Icons

Components
----------
WeatherCard: Current conditions
ForecastSection: 5-day forecast
AirQualityCard: Air quality data
WeatherAlert: Weather warnings
WeatherImpact: Activity suggestions

Updates
-------
Weather data refreshes every 5 minutes
Air quality updates in real-time
